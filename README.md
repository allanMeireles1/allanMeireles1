# 🔍 Allan Meireles

**`QA Júnior`**

Me chamo Allan Meireles, tenho 21 anos e estou em transição para a área de Quality Assurance. Atualmente venho estudando QA de forma prática, com foco em testes manuais, validação de APIs e automação, construindo projetos reais documentados no GitHub.

Tenho me dedicado ao desenvolvimento de casos de teste estruturados, execução de testes exploratórios, documentação de bugs com evidências e validação de APIs utilizando Postman e SQL. Todo o meu progresso está sendo registrado em um projeto público onde documento a evolução técnica semana a semana.

Meu objetivo é atuar como QA Júnior ou estagiário, contribuindo para a qualidade do produto desde as primeiras etapas de desenvolvimento e evoluindo continuamente como profissional de qualidade de software.
